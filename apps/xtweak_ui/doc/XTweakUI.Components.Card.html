    XTweakUI.Components.Card (xtweak_ui v0.1.0)

    Card component for Phoenix LiveView applications.

    Provides a flexible card layout with header, body, and footer sections.

    Summary

    Functions

    Renders a card component.

    Examples

    <.card>
  <:header>Card Title</:header>
  <:body>Card content goes here</:body>
  <:footer>
    <button>Action</button>
  </:footer>
</.card>

    Attributes

    • class (:string) - Defaults to "".
    • shadow (:boolean) - Defaults to true.

    Slots

    • header
    • body (required)
    • footer
