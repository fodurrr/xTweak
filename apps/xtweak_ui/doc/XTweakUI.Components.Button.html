    XTweakUI.Components.Button (xtweak_ui v0.1.0)

    Button component with multiple variants, sizes, and states.

    Ported from Nuxt UI's UButton component.

    Examples

    # Basic button
<.button>Click me</.button>

# With color and variant
<.button color="primary" variant="solid">
  Submit
</.button>

# With size
<.button size="lg">Large Button</.button>

# Loading state
<.button loading>
  Submit
</.button>

# With icon
<.button icon="hero-arrow-right">
  Next
</.button>

# Disabled
<.button disabled>
  Disabled
</.button>

# Block (full width)
<.button block>
  Full Width
</.button>

    Accessibility

    • Keyboard navigable (Tab, Enter/Space)
    • ARIA disabled state when disabled
    • Focus visible styles
    • Screen reader compatible

    Attributes

    • label (:string) - Button label (alternative to inner_block). Defaults to nil.
    • color (:string) - Button color (primary, secondary, success, warning, error, neutral). Defaults to "primary".
    • variant (:string) - Visual variant (solid, outline, soft, ghost, link). Defaults to "solid".
    • size (:string) - Button size (xs, sm, md, lg, xl). Defaults to "md".
    • loading (:boolean) - Show loading spinner. Defaults to false.
    • disabled (:boolean) - Disable button. Defaults to false.
    • icon (:string) - Leading icon (Heroicons name, e.g., 'hero-check'). Defaults to nil.
    • leading_icon (:string) - Alias for icon (leading icon). Defaults to nil.
    • trailing_icon (:string) - Trailing icon. Defaults to nil.
    • block (:boolean) - Full width button. Defaults to false.
    • square (:boolean) - Square button (equal padding). Defaults to false.
    • type (:string) - Button type (button, submit, reset). Defaults to "button".
    • class (:string) - Additional CSS classes. Defaults to "".
    • Global attributes are accepted. Additional HTML attributes. Supports all globals plus: ["phx-click", "phx-target", "form", "name", "value", "href", "target"].

    Slots

    • inner_block - Button content.
